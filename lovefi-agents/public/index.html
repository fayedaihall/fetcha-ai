LoveFi Agents API

This is the LoveFi dating matcher agent API.

Available API endpoints:

  • /api/test - ✅ Fixed test endpoint (working)
  • /api/simple - ✅ Simple compatibility analyzer (working)
  • /api/index - Advanced compatibility analyzer (FastAPI)

API Usage

For compatibility analysis, send a POST request to /api/simple with:

{
  "profile1": {
    "age": 25,
    "interests": ["hiking", "music"],
    "location": "New York"
  },
  "profile2": {
    "age": 27,
    "interests": ["hiking", "art"],
    "location": "New York"
  }
}
